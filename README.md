# projets_uni
Contient tous les projets créés durant mes années à l'UBO de Brest.

-----------------------------------------------------------------------------------------
# projet.environnement
Projet web de L1, dans lequel on devait créé un site web qui traitait d'un problème environnemental, que ce soit réel ou fictif. J'ai choisi de traiter la planète fictive de Mandalore (planète de l'univers Star Wars).

-----------------------------------------------------------------------------------------
# projet.pokemon
Projet web de L2 semestre 1, dans lequel on devait créer un jeu/quiz de pokémon sur un site web en untilisant que des fonctionalités côté client. Nécessite d'être connecté au réseau de l'UBO afin de pouvoir accéder aux données sur le fichier XML.

-----------------------------------------------------------------------------------------
# projet.sellection
Projet web de L2 semestre 2, dans lequel on devait créer une vitrine pour une organisation ou un thème de notre choix (mon choix : marchandise BTS) en utilisant des fonctionnalités de web côté serveur. Nécessite d'être connecté au réseau de l'UBO afin de pouvoir accéder aux données sur le serveur. Contient 2 versions (à la demande de notre professeure), la v2 étant la version la plus à jour.
Lien pour la version la plus à jour : https://obiwan.univ-brest.fr/~e22400857/v2/index.php
